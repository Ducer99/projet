# 🎨 Guide des Couleurs par Genre - Arbre Généalogique

## ✅ Améliorations Appliquées

### 🔵 **Garçons/Hommes (Masculin)**
- **Couleur de fond** : Bleu très clair (`blue.50`)
- **Bordure** : Bleu (`blue.300` normal, `blue.500` en focus)
- **Icône** : ♂ en bleu (`blue.600`)
- **Badge genre** : Badge bleu solide "♂ HOMME"
- **Avatar** : Arrière-plan bleu (`#4299e1`)

### 🔴 **Filles/Femmes (Féminin)**
- **Couleur de fond** : Rose très clair (`pink.50`)
- **Bordure** : Rose (`pink.300` normal, `pink.500` en focus)  
- **Icône** : ♀ en rose (`pink.600`)
- **Badge genre** : Badge rose solide "♀ FEMME"
- **Avatar** : Arrière-plan rose (`#ed64a6`)

### ⚪ **Genre Inconnu**
- **Couleur de fond** : Blanc ou gris très clair
- **Bordure** : Gris (`gray.300`)
- **Avatar** : Arrière-plan gris (`#718096`)
- **Pas d'icône de genre**

## 🏡 **Groupes Familiaux**

### **Conjoints**
- Fond coloré selon le genre du conjoint
- Bordure assortie pour créer une harmonie visuelle

### **Enfants par Union**
- Chaque enfant a sa couleur selon son genre
- Les groupes d'enfants conservent un fond neutre pour la lisibilité

## 🎯 **Comment Tester**

1. **Ouvrir l'arbre généalogique** : `http://localhost:3002`
2. **Observer les cartes** : 
   - Les hommes ont des tons bleus
   - Les femmes ont des tons roses
   - Les bordures sont plus prononcées pour la personne en focus
3. **Navigation** :
   - Cliquer sur une personne pour la mettre en focus
   - Observer comment les couleurs se maintiennent
4. **Sections spéciales** :
   - **Conjoints** : Voir les couleurs par genre dans la colonne de gauche
   - **Enfants** : Voir la distinction garçons/filles dans la colonne de droite
   - **Frères/sœurs** : Voir les couleurs dans la section du haut

## 🎨 **Avantages de cette Approche**

### ✅ **Clarité Visuelle**
- Distinction immédiate entre garçons et filles
- Particulièrement utile pour les familles nombreuses
- Les enfants sont facilement identifiables par genre

### ✅ **Consistance**
- Même système de couleurs dans toute l'application
- Icônes (♂/♀) et couleurs concordantes
- Badges informatifs avec couleurs assorties

### ✅ **Accessibilité**
- Couleurs subtiles mais perceptibles
- Compatible avec les modes clair/sombre
- Multiple indicateurs visuels (couleur + icône + badge)

### ✅ **Généalogie Améliorée**
- Facilite la compréhension des liens familiaux
- Aide à identifier rapidement les lignées
- Particulièrement utile pour analyser les mariages et descendances
