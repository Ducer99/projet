# 🌳 Guide Visuel Rapide - Arbre Organique v2

## 🎨 Ce que vous allez voir

### 1. **Cadres Colorés et Ondulés**

```
╭─────────────────╮
│   ╭───────╮     │  ← Bordure ondulée (non rectangulaire)
│   │ Photo │     │  ← Photo circulaire
│   ╰───────╯     │
│                 │
│   Jean Dupont   │  ← Nom en gras
│   (1950-2023)   │  ← Dates/Âge
╰─────────────────╯
  Couleur vive variée (rose, jaune, vert, bleu...)
```

### 2. **Couples Côte à Côte**

```
╭──────────╮  💍💍  ╭──────────╮
│ Grand-   │ ═══════│ Grand-   │
│ Père     │  OR    │ Mère     │
╰──────────╯ (mariage) ╰──────────╯
     │                    │
     └─────────┬──────────┘
               │ (filiation)
        ╭──────▼──────╮
        │   Enfant    │
        ╰─────────────╯
```

### 3. **Branches d'Arbre**

```
        Racine
          │
          │ ← Épais, marron
         / \
        /   \ ← Moins épais
       /     \
   Enfant1  Enfant2 ← Encore moins épais
```

### 4. **Distinction Mariage vs Filiation**

**Ligne de Mariage :**
- Couleur : 💛 **OR** (#FFD700)
- Style : `─ ─ ─` **Pointillés**
- Direction : **Horizontal**
- Symbole : 💍 **Anneaux entrelacés**

**Ligne de Filiation (Parent→Enfant) :**
- Couleur : 🌰 **Marron** (#8B4513)
- Style : `─────` **Continu**
- Direction : **Vertical/Courbe**
- Épaisseur : **Variable**

## ✨ Caractéristiques Visuelles

### Cadres
- ✅ Bordures **ondulées** (pas rectangulaires)
- ✅ Couleurs **vives et variées**
- ✅ Ombre portée pour **effet 3D**
- ✅ Double bordure pour **effet de profondeur**

### Branches
- ✅ Lignes **courbes naturelles**
- ✅ Couleur **bois** (#8B4513)
- ✅ Épaisseur **variable** (plus épais à la racine)
- ✅ Variations **aléatoires** pour look organique

### Mariages
- ✅ Ligne **horizontale** en or
- ✅ Style **pointillé**
- ✅ Symbole **💍 anneaux entrelacés**
- ✅ **Clairement distingué** des lignes parent-enfant

### Conjoints
- ✅ Affichés **côte à côte**
- ✅ **Même hauteur** verticale
- ✅ Cadres **identiques** aux autres
- ✅ Couleurs **complémentaires**
