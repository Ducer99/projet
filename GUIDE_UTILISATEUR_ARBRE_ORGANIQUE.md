# 🎨 Vue d'Ensemble : Arbre Généalogique Organique

## 🌳 Qu'est-ce que c'est ?

Une **visualisation d'arbre généalogique esthétique et interactive** qui remplace les boîtes rigides par une représentation organique avec :

- 🌲 **Tronc d'arbre** à la racine
- 🌿 **Branches courbes** (courbes de Bézier)
- 🍃 **Feuilles décoratives** dispersées
- 🖼️ **Cadres vintage** pour les photos
- 🎨 **Couleurs distinctives** selon sexe et statut

---

## 🚀 Comment y Accéder ?

### Méthode 1 : Via le Menu (Recommandé)
1. Connectez-vous à l'application
2. Dans le header, cliquez sur **"Arbre"**
3. Sélectionnez **"🌳 Vue Organique"**

### Méthode 2 : URL Directe
```
http://localhost:3000/family-tree-organic
```

### Méthode 3 : Avec Focus sur une Personne
```
http://localhost:3000/family-tree-organic?focus=24
```
*(Remplacez `24` par l'ID de la personne)*

---

## 🎮 Comment Utiliser ?

### Navigation

| Action | Méthode | Résultat |
|--------|---------|----------|
| **Zoom** | Molette souris ou Boutons 🔍 | Agrandir/Réduire la vue |
| **Déplacer** | Glisser avec la souris | Naviguer dans l'arbre |
| **Voir profil** | Click sur une personne | Ouvre le profil détaillé |
| **Recentrer** | Double-click ou Bouton 🏠 | Recentre sur une personne |
| **Vue standard** | Bouton 📊 | Bascule vers vue classique |

### Boutons de Contrôle

Dans le coin supérieur droit, vous trouverez :

- 🏠 **Recentrer** - Réinitialise la vue
- 🔍+ **Zoom avant** - Agrandit la vue
- 🔍- **Zoom arrière** - Réduit la vue
- 📊 **Vue Standard** - Bascule vers l'arbre classique

### Bordures des Cadres

| Couleur | Signification |
|---------|---------------|
| 🔵 **Bleu** | Homme |
| 🌸 **Rose** | Femme |

- ⭐ **Badge doré** - Personne racine de l'arbre

- 🌫️ **Overlay gris** - Personne décédée

## 🆚 Vue Organique vs Vue Standard

| Critère | Vue Standard | Vue Organique |
|---------|--------------|---------------|
| **Style** | Boîtes rectangulaires | Cadres vintage + arbre |
| **Liens** | Lignes droites | Branches courbes |
| **Décoration** | Minimaliste | Tronc, feuilles, dégradé |
| **Zoom** | Basique | Avancé (boutons + molette) |
| **Best for** | Données précises | Expérience visuelle |

**Notre recommandation :** Utilisez la **vue organique** pour explorer et la **vue standard** pour analyser.

## 🐛 Problèmes Courants

### L'arbre ne s'affiche pas
- ✅ Vérifiez que vous êtes connecté
- ✅ Vérifiez que votre famille a des membres
- ✅ Rafraîchissez la page (F5)

### Les photos ne s'affichent pas
- ✅ Vérifiez les URLs des photos dans les profils
- ✅ Un avatar par défaut est utilisé si pas de photo

### Le zoom ne fonctionne pas
- ✅ Essayez les boutons 🔍 au lieu de la molette
- ✅ Cliquez sur 🏠 pour réinitialiser

### Performance lente
- ✅ Normal si >100 personnes dans l'arbre
- ✅ Fermez les autres onglets
- ✅ Utilisez Chrome ou Firefox (meilleure perf)
